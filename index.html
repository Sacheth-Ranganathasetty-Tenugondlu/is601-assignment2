This is the second assignment of the course to learn how to host websites using GitHub Pages by using linux commands, Vi Text Editor and Git.

Linux Commands

  • CD - Change Directory
  • MKDIR - Make Directory
  • LS - List Files

Vi Commands

  • I - Insert mode
  • Esc - Get out of insert mode
  • Shift key + Colon - Command Prompt for vi
  • W - Command to write
  • Q - Command to quit
  • WQ - Write and Quit

Git Commands

  • Git init - Create Repository
  • Git add Filename - Add file to git
  • Git commit -m "message" - Do the commit to save changes
  • Git checkout -b BranchName - Make a new branch
  • Git merge BranchName - Merge the branch
